# 🛒 Smart Shopping Dashboard

An interactive shopping dashboard built using **Python** and **Streamlit**.  
This project demonstrates core **Object-Oriented Programming (OOP)** concepts like abstraction, inheritance, and modular design.

---

## 🚀 Features

- 📂 Category-based product selection (Electronics, Clothing, Groceries)
- 🛍️ Dynamic product listing
- 💰 Automatic price calculation
- 🎉 Flat 10% discount on all products
- 🧾 Bill generation system
- 🎨 Clean and interactive UI using Streamlit

---

## 🧠 Concepts Used

- Abstract Classes (`ABC`)
- Inheritance
- Polymorphism (method overriding)
- Modular programming
- Streamlit UI components

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🎈
